
## Recommended IDE Setup

# Vue 3 + Laravel 11 + SQLite Project

## vue Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

## laravel Project Setup

```sh
composer install

```

### migrate the database

```sh
php artisan migrate
```

### Compile and Minify for Production 

```sh
php artisan serve
```
